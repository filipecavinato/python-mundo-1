# 🐍 Curso de Python - Curso em Vídeo

Repositório dedicado aos exercícios e desafios propostos pelo professor Gustavo Guanabara no Curso em Vídeo. O objetivo aqui é praticar lógica de programação, sintaxe e estruturas de dados.

## 📂 Sobre este Módulo

Este repositório contém as resoluções dos exercícios propostos no **Mundo 1** do Curso em Vídeo, onde o foco é o aprendizado dos fundamentos iniciais da linguagem:
* **Instalação e Configuração:** Preparação do ambiente de desenvolvimento.
* **Tipos Primitivos:** Inteiros, floats, strings e booleanos.
* **Operadores Aritméticos:** Cálculos matemáticos básicos.
* **Utilizando Módulos:** Importação de bibliotecas básicas (math, emoji, etc).

## 🚀 Como executar

Como são scripts individuais, cada arquivo .py pode ser executado diretamente pelo terminal:

```
python pasta_do_exercicio/nome_do_arquivo.py
```

## 🛠️ Tecnologias

Linguagem: Python 3.x
Ambiente: PyCharm
Bibliotecas: `random`, `time`, `datetime`, `rich`, `os`
Versionamento: Git & GitHub
